# Hardware Profiles

Hardware profile workspace for supported ground nodes, radios, antennas, cables,
and deployment configurations.

Known blocked items:

- Digi A09-F8NF-M antenna gain and radiation pattern.
- Nearson S463AH-915 antenna gain, radiation pattern, and ground-plane
  requirement.
- Cable and connector loss values.

Do not fill these values until datasheets or authoritative sources are verified.
